# Predator-Prey Simulation with Reproductive Latency

To install locally:
```
git clone blah blah

# or to update

git pull
```


Then:
```
python3 -m venv .venv
```

For linux/macOS:
```
source .venv/bin/activate
```

for windows:
```
.venv/Scripts/activate
```

after activating .venv:
```
pip install -r requirements.txt
```
